# WASM Port: Rich Man Game Engine → Rust

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Port all game logic from inline JS in `index.html` to a Rust WASM module, keeping UI/DOM rendering in JS.

**Architecture:** Rust `wasm-bindgen` crate compiled to `.wasm` via `wasm-pack`. JS loads the WASM module, calls into Rust for all game state mutations and calculations, then renders the returned JSON as HTML. Single `index.html` still deploys to GitHub Pages, but now loads `pkg/rich_man_engine.js` + `.wasm`.

**Tech Stack:** Rust 1.97, `wasm-bindgen`, `serde`/`serde_json`, `js-sys` (Math.random), `wasm-pack`, GitHub Pages

```rust
use serde::{Serialize, Deserialize};

pub fn js_random() -> f64 {
    js_sys::Math::random()
}

pub fn random_range(min: f64, max: f64) -> f64 {
    min + js_random() * (max - min)
}

pub fn roll_d20() -> u32 {
    (js_random() * 20.0).floor() as u32 + 1
}

pub fn random_int(min: f64, max: f64) -> i32 {
    (random_range(min, max)).round() as i32
}

#[derive(Serialize, Deserialize, Clone, Debug)]
pub struct RollOutcome {
    pub tier: String,
    pub label: String,
    pub multiplier: f64,
    pub effective_roll: i32,
}

pub fn roll_outcome(roll: u32, bonus: i32) -> RollOutcome {
    let effective = (roll as i32 + bonus).max(1).min(20);
    let (tier, label, multiplier) = if effective >= 17 {
        ("great", "대성공 🔥", random_range(2.0, 2.5))
    } else if effective >= 13 {
        ("good", "성공 📈", random_range(1.3, 1.5))
    } else if effective >= 8 {
        ("moderate", "보통 📊", random_range(0.9, 1.1))
    } else if effective >= 4 {
        ("poor", "부진 😓", random_range(0.5, 0.7))
    } else {
        ("critical", "위기 😱", random_range(0.1, 0.3))
    };
    RollOutcome {
        tier: tier.into(),
        label: label.into(),
        multiplier,
        effective_roll: effective,
    }
}
```

```rust
pub fn pick_random_event(category: &str, state: &GameState) -> Option<(EventResult, Box<dyn EventScenario>)> {
    let all = get_all_events();
    let prefix = match category {
        "re" => "re_", "sc" => "sc_", "bz" => "bz_",
        "pt" => "pt_", "lf" => "lf_", _ => "",
    };
    let candidates: Vec<&Box<dyn EventScenario>> = all.iter()
        .filter(|e| e.id().starts_with(prefix) && e.trigger(state))
        .collect();
    if candidates.is_empty() { return None; }

            let flavor = e.flavor(state);

            for (i, owned) in all.iter().enumerate() {
```

```rust
pub fn generate_property() -> Property {
    let area = &TOKYO_AREAS[(js_random() * TOKYO_AREAS.len() as f64).floor() as usize];
    let price = (random_range(area.base_price[0], area.base_price[1]) / 10.0).round() * 10.0;
    let ltv = random_range(0.75, 0.85);
    let loan = (price * ltv / 10.0).round() * 10.0;
    let down_payment = price - loan;
    let rr = random_range(area.rent_rate[0], area.rent_rate[1]);
    let mr = (price * rr / 12.0 * 10.0).round() / 10.0;
    let mf = mr * 0.15;
    let mi = (loan * 0.0075) / 12.0;

        prop_type: "1K".into(),
```

```rust
pub fn process_exit(state: &GameState) -> ExitResult {
    let mut breakdown = ExitBreakdown::default();

    // Business sale
    if state.business.active && state.business.value > 0.0 {
        let sv = (state.business.value * state.business.stake).round();
        let bt = (sv * 0.20).round();
        breakdown.business = Some(sv - bt);
    }

    // Real estate
    let mut re_total = 0.0;
    for p in &state.real_estate {
        let cv = (p.purchase_price * (1.0 + p.value_change)).round();
        let gain = (cv - p.purchase_price).max(0.0);
        let tx = (gain * 0.15).round();
        re_total += cv - p.loan - tx;
    }
    breakdown.real_estate = re_total;

    // Stocks
    let q_gain = (state.stocks.qqq - state.stocks.qqq_cost).max(0.0);
    let q_tx = (q_gain * 0.20315).round();
    breakdown.qqq = state.stocks.qqq - q_tx;

    let c_gain = (state.stocks.crypto - state.stocks.crypto_cost).max(0.0);
    let c_tx = (c_gain * 0.20315).round();
    breakdown.crypto = state.stocks.crypto - c_tx;

    // Cash + emergency (no tax)
    breakdown.cash = state.cash + state.emergency;

    // Business cash (dividend 20%)
    breakdown.business_cash = if state.business.cash > 0.0 {
        let dct = (state.business.cash * 0.20).round();
        state.business.cash - dct
    } else { 0.0 };

    // PropTech
    breakdown.proptech = if state.proptech.active && state.proptech.value > 0.0 {
        let ptt = (state.proptech.value * 0.20).round();
        state.proptech.value - ptt
    } else { 0.0 };

    let total = breakdown.business.unwrap_or(0.0) + breakdown.real_estate
        + breakdown.qqq + breakdown.crypto + breakdown.cash
        + breakdown.business_cash + breakdown.proptech;

    ExitResult {
        breakdown,
        total_after_tax: total.round(),
    }

}

pub fn generate_share_text(state: &GameState, final_total: f64) -> String {
    let gm = if state.start_capital > 0.0 {
        format!("{:.1}", final_total / state.start_capital)
    } else { "∞".into() };
    let rank = get_rank(final_total);
    let sn = match state.strategy.as_str() {
        "realestate" => "부동산 올인",
        "stocks" => "주식/암호 올인",
        "business" => "창업 올인",
        _ => "복합 전략",
    };
    format!("{}에서 {}까지 {}년간 {}배 성장!\n전략: {}\n랭크: {} {}\n#20億円の夢",
        format_yen(state.start_capital), format_yen(final_total),
        state.duration, gm, sn, rank.emoji, rank.name)
}
```

Tests to write:

6. `test_roll_outcome_tiers` — verify all 5 tiers with mocked rolls
7. `test_get_rank` — verify all rank tiers
8. `test_process_exit_tax` — verify tax calculations

Since `js_sys::Math::random()` requires a browser, tests need either:
- A `getrandom` crate with `js` feature for WASM tests, OR
- Use `cfg(test)` with a deterministic RNG replacement

**Approach:** Make RNG injectable via a trait, so tests can use a seeded RNG:

```rust
// In rng.rs, add a trait:
pub trait Rng: Send + Sync {
    fn random(&self) -> f64;
}

// Default impl using js_sys:
pub struct JsRng;
impl Rng for JsRng {
    fn random(&self) -> f64 { js_sys::Math::random() }
}

// Test impl:
#[cfg(test)]
pub struct SeededRng { state: u64 }
#[cfg(test)]
impl Rng for SeededRng {
    fn random(&self) -> f64 {
        // xorshift64
        let mut x = self.state;
        x ^= x << 13; x ^= x >> 7; x ^= x << 17;
        self.state = x;
        (x as f64) / (u64::MAX as f64)
    }
}
```

Then all functions take `&dyn Rng` instead of calling `js_sys::Math::random()` directly. This makes the engine fully testable.
